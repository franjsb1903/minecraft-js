# Minecraft Clone JS

Clon de Minecraft realizado en JavaScript basado en el vídeo de [Midudev](https://www.youtube.com/watch?v=dm7nfe3bOE4) y [FreeCodeCamp](https://www.youtube.com/watch?v=qpOZup_3P_A&t=4030s).

## Teconologías

- [React](https://reactjs.org/) 
- [ThreeJS](https://threejs.org/)
  - [@react-three/cannon](https://cannon.pmnd.rs/) 
  - [@react-three/drei](https://drei.pmnd.rs/)
  - [@react-three/fiber](https://docs.pmnd.rs/react-three-fiber/)
- [Zustand](https://github.com/pmndrs/zustand)
- [ViteJS](https://vitejs.dev)
